# Travel Website 🌍

## About The Project
This is a responsive travel website built using HTML, CSS, and JavaScript.  
It helps users explore travel destinations, view packages, and get travel-related information.

## Features
- Responsive design (mobile + desktop)
- Beautiful landing page
- Travel destination sections
- Smooth navigation 

## Technologies Used
- HTML
- CSS3
- JavaScript

## Project Structure
- index.html
- style.css
- script.js

## How to Run
1. Download or clone this repository
2. Open `landing 2 .html` in any browser
3. Explore the website
